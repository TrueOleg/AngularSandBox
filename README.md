# AngularSandBox

Template to start mobile application development on ionic 4 with firebase integration

## Getting Started

```
git clone https://github.com/TrueOleg/AngularSandBox.git
```

### Prerequisites

```
nodejs version 10.15.0
```

```
TypeScript 3.4.5
```

### Installing

```
npm install
```

### Run

```
npm start
```
