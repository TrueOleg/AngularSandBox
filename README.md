# AngularSandBox

Angular Test Task - improvement of the application that has an entry page with 3 links to 3 different forms.

## Getting Started

```
git clone https://github.com/TrueOleg/AngularSandBox.git
```

### Prerequisites

```
nodejs version 10.15.0
```

```
TypeScript 3.4.5
```

### Installing

```
npm install
```

### Run

```
npm start
```
